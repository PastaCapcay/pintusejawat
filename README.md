# PintuSejawat - Medical Learning Platform

## 🇬🇧 English

### Overview
PintuSejawat is a comprehensive medical learning platform designed to help medical students and professionals enhance their knowledge through practice questions and assessments.

### Features
- **Practice Questions**
  - Multiple choice questions with detailed explanations
  - Real-time scoring and performance tracking
  - Instant feedback for each answer
  - Comprehensive explanations for correct answers

- **Exam Simulation**
  - Timed exam simulations
  - Various medical topics coverage
  - Score tracking and performance analytics
  - Review completed exams with detailed feedback

- **Performance Analytics**
  - Score history tracking
  - Progress visualization through graphs
  - Identify strengths and areas for improvement
  - Compare performance across different topics

- **User Experience**
  - Clean and intuitive interface
  - Mobile-responsive design
  - Easy navigation between questions
  - Progress saving and resume capability

### Technology Stack
- **Frontend**
  - Next.js 14 (App Router)
  - TypeScript
  - Tailwind CSS
  - Shadcn UI Components

- **Backend**
  - Next.js API Routes
  - Prisma ORM
  - PostgreSQL Database

- **Authentication**
  - NextAuth.js
  - Secure session management

### Recent Updates
- Added practice question history
- Improved exam review interface
- Enhanced performance analytics with graphs
- Updated UI/UX for better user experience
- Added detailed explanations for each question

## 🇮🇩 Indonesia

### Ikhtisar
PintuSejawat adalah platform pembelajaran medis yang dirancang untuk membantu mahasiswa dan profesional medis meningkatkan pengetahuan mereka melalui latihan soal dan penilaian.

### Fitur
- **Latihan Soal**
  - Soal pilihan ganda dengan penjelasan detail
  - Penilaian langsung dan pelacakan performa
  - Umpan balik instan untuk setiap jawaban
  - Penjelasan komprehensif untuk jawaban yang benar

- **Simulasi Ujian**
  - Simulasi ujian dengan batasan waktu
  - Mencakup berbagai topik medis
  - Pelacakan nilai dan analisis performa
  - Review ujian yang telah selesai dengan umpan balik detail

- **Analisis Performa**
  - Pelacakan riwayat nilai
  - Visualisasi kemajuan melalui grafik
  - Identifikasi kekuatan dan area yang perlu ditingkatkan
  - Bandingkan performa di berbagai topik

- **Pengalaman Pengguna**
  - Antarmuka yang bersih dan intuitif
  - Desain responsif untuk mobile
  - Navigasi mudah antar soal
  - Kemampuan menyimpan dan melanjutkan progres

### Teknologi yang Digunakan
- **Frontend**
  - Next.js 14 (App Router)
  - TypeScript
  - Tailwind CSS
  - Komponen Shadcn UI

- **Backend**
  - Next.js API Routes
  - Prisma ORM
  - Database PostgreSQL

- **Autentikasi**
  - NextAuth.js
  - Manajemen sesi yang aman

### Pembaruan Terkini
- Penambahan riwayat latihan soal
- Peningkatan antarmuka review ujian
- Peningkatan analisis performa dengan grafik
- Pembaruan UI/UX untuk pengalaman pengguna yang lebih baik
- Penambahan penjelasan detail untuk setiap soal
